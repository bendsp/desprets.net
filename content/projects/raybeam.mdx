[RayBeam](https://github.com/bendsp/RayBeam) is a configurable C++ raytracer I built with classmates during my third
year of CS.

It renders simple scenes from config files, with spheres, planes, cones,
cylinders, point lights, directional lights, reflections, shadows, and a small
SFML window for moving the camera around.

The code is very much a student project, but I still like what it represents:
learning graphics from first principles, building the math by hand, and getting
something visual on screen without a game engine doing the work for us.

I keep the original 2023 version available as
[v0.1.0](https://github.com/bendsp/RayBeam/releases/tag/v0.1.0).
